# AI Daily Digest

Automated daily digest of AI industry news and notable research papers, published via GitHub Pages.

- `index.html` — running list of days, newest first
- `digests/YYYY-MM-DD.html` — one page per day's digest
- `seen.json` — dedup manifest (normalized URL/title → date first covered), used to avoid repeating stories already covered in the last ~14 days

Generated daily by a scheduled agent run. Not manually edited.
